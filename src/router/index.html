CASH2GO

A loan prediction app


API Documentation:


User Endpoints:
https://cash2go-backendd.onrender.com/api/v1/user

POST /send-otp: Sends an OTP verification email to the specified email address.

PATCH /resend-otp: Resends an OTP verification email to the specified email address.

PATCH /signup: Creates/Updates a new user account with the provided details.

PATCH /verify-otp: Verifies the OTP sent to the user's email address.

PATCH /security-question: Saves the security question and answer for the user.

POST /login: Logs in the user with the provided email and password.

GET /search: Searches for a user by their username.

POST /logout: Logs out the user.

GET /get-user-info/:id: Retrieves the user information for the specified user ID.

POST /edit-user-info/:id: Edits the user information for the specified user ID.

POST /verify-email: Verifies the user's email address before initiating password reset.

GET /get-security-question: Get the security question for password reset.

PATCH /reset-password: Verifies the userâ€™s answer to the security question and sends a reset token.

POST /update-password/:token: Update the user's password using a reset token.


Applicant Endpoints:
https://cash2go-backendd.onrender.com/api/v1/applicant

GET /contact/:id: Retrieves the contact information for the specified applicant ID.

GET /prediction/:id: Retrieves the prediction information for the specified applicant ID.

GET /approved-applicants: Retrieves the list of approved applicants.

GET /pending-applicants: Retrieves the list of pending applicants.

GET /rejected-applicants: Retrieves the list of rejected applicants.

GET /applicants: Retrieves the list of all applicants.

Messaging Endpoints:
https://cash2go-backendd.onrender.com/api/v1/messaging

POST /send-message: Sends messages to applicants.

GET /get-messages: Get all messages sent to applicants.
